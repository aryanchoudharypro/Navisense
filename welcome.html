Welcome to Navisense

An AI-powered accessibility-first browser designed especially for screen reader users.

Navisense is an experimental accessibility-focused browser that improves the usability of websites for visually impaired users using Artificial Intelligence. What Navisense does: • Scans web pages after loading • Detects accessibility issues automatically • Fixes unlabeled buttons, inputs, and controls • Improves screen reader announcements • Allows live accessibility adjustments without reloading pages How to get started: 1. Type a website address in the address bar and press Enter. 2. Use the Accessibility menu to: - Toggle password visibility - Enable or disable dark mode - Adjust text size 3. Navigate using standard keyboard keys. (Browse Mode may be limited due to embedded browser constraints.) Important accessibility note: This content is presented inside an editable reading area so that screen readers like NVDA can read it correctly even when browse mode is unavailable. Navisense is a research and learning project created to explore how AI can improve real-world web accessibility.
